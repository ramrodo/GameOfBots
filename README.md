# Bot Name

Game Of Bots - Hackaton

## Installation

## Ubuntu

$ sudo apt-get install golang
$ go get -u -v github.com/Recime/recime-cli

## Archlinux

$ sudo pacman -S go
$ go get -u -v github.com/Recime/recime-cli

## Create symbolic link

sudo ln -s /home/ramrodo/go/bin/recime-cli /usr/bin/recime-cli

## Usage

$ recime-cli login
$ recime-cli deploy
